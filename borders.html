HTML Borders:

The HTML <table> border Attribute is used to specify the border of a table.
It sets the border around the table cells.

Supported elements

HTML border attribute supports table element.

Syntax:

<table border="value">.......</table>

Attributes Value

1: It sets the border around the table cells.
0: It removes (not set) the border around the table cells.

Default Value

There is no default value of HTML border attribute

Example of HTML border with attribute value 1

Example:

<table border="1">
<caption>Author Details</caption>
<tr>
<th>Name</th>
<th>Age</th>
<th>RollNo</th>
</tr>
<tr>
<td>Nabanita Dutta</td>
<td>23</td>
<td>IT-12</td>
</tr>
<tr>
<td>Chitralekha Paul</td>
<td>23</td>
<td>IT-03</td>
</tr>
</table>

Output:

Author Details
Name Age RollNo
Nabanita Dutta 23 IT-12
Chitralekha Paul 23 IT-03
